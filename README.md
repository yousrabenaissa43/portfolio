# Project Name

## Overview
This project is a web application developed using React and TypeScript.
The initial structure was generated using Lovable, then significantly modified to add custom features and improve the UI.

## Features
- Modern responsive UI
- Component-based architecture
- Custom business logic and workflows

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
